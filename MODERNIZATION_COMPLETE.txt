💡 CUSTOMIZATION TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Change Colors:
  1. Edit app/globals.css
  2. Update --primary, --accent, etc. variables
  3. All pages automatically update

Modify Icons:
  1. Import from lucide-react
  2. Replace icon in component
  3. Adjust size with w-X h-X classes

Adjust Spacing:
  1. Use Tailwind py-X sm:py-X lg:py-X classes
  2. Adjust gap-X values in grids
  3. Modify px-X for horizontal padding

Add Features:
  1. Create new component in components/
  2. Use existing components as base
  3. Follow design system
