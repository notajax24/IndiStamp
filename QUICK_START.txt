🚀 QUICK START

1. Install & Run:
   $ cd /vercel/share/v0-project
   $ pnpm dev
   ➜ Open http://localhost:3000

2. Test Sample Pages:
   - Home: http://localhost:3000/
   - Maharashtra: http://localhost:3000/stamp-duty-calculator/maharashtra
   - Delhi: http://localhost:3000/stamp-duty-calculator/delhi
   - Karnataka: http://localhost:3000/stamp-duty-calculator/karnataka

3. Update Rates:
   - Edit: data/stampDutyRates.ts
   - Modify stampDutyRate, registrationRate, concessions
   - All pages auto-update on save

4. Build & Deploy:
   $ pnpm build
   $ vercel deploy

🔧 COMMON TASKS

Update Stamp Duty Rates:
1. Edit data/stampDutyRates.ts
2. Modify stampDutyRate field
3. Save → all pages auto-update

Add State Notes:
1. Edit data/stampDutyRates.ts
2. Update notes field
3. Notes appear on state page

Enable AdSense:
1. Get AdSense publisher ID
2. Replace <div id="ad-slot"> divs
3. Locations: home page + all state pages

Add New State (future):
1. Add entry to stampDutyRates.ts
2. Set state, slug, rates, concessions
3. Run `pnpm build` → page auto-generates
